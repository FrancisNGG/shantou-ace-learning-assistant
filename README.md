# 汕大成考学习助手

本项目是一个基于 Electron 的汕头大学继续教育学院在线学习自动化辅助工具，支持课程列表展示、自动播放等功能。

## 功能特色

- 自动登录与记住密码
- 课程列表与成绩展示
- 自动播放课程视频
- 章节进度追踪
- 日志记录与导出
- 支持开发者模式

## 使用方法

1. 安装依赖

   ```bash
   npm install
   ```

2. 启动应用

   ```bash
   npm start
   ```

3. 打包应用

   ```bash
   npm run dist
   ```

## 依赖环境

- Node.js
- Electron
